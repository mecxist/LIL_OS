# LIL OS (v0.1.1)

LIL OS is a constitutional substrate for human–AI systems: it governs how goals are formed, challenged, revised, and revoked — not just how agents execute tasks.

## Required Reading
- GOVERNANCE.md
- CONTEXT_BUDGET.md
- RESET_TRIGGERS.md
- DECISION_LOG.md
- RULE_IDS.md

## Intent-Level Change Rule
Any change to goals, metrics, agent autonomy, `.cursorrules`, or non-negotiables is an **intent-level change** and MUST:
1) follow GOVERNANCE.md, and
2) be logged in DECISION_LOG.md.
